1. Парсинг арифметических выражений реализован для файлов следующих типов:
-JSON
-XML
-PLAIN
примеры для входных данных этих файлов можно найти в папке Assets

2. Архиватор - ZIP, Шифровальный алгоритм - DES

3. Вложенность архивов может быть любая, но зашифрован может быть только конечный файл(именно файл, не архив)

4. WebUI реализован с помощью сервиса Bootstrap Studio через localhost:8080

5. Присутсвует Rest Web Service
